# Data Cleaning & Preprocessing (Task 1)

## 📌 Project Overview
This repository contains the work for **Task 1** of the Data Science Internship at **Synent Technologies**. The goal of this project is to clean and preprocess raw data to make it ready for exploratory data analysis (EDA) and machine learning models.

---

## 🛠️ Key Steps & Methodology

1. **Data Inspection & Exploration**
   * Loaded raw datasets and checked basic summary statistics.
   * Identified missing values, duplicate entries, and incorrect data types.

2. **Data Cleaning**
   * **Missing Values:** Imputed or removed missing values based on data distribution.
   * **Duplicates:** Removed redundant records.
   * **Outliers:** Handled extreme values using IQR / Z-score techniques.
   * **Data Formatting:** Standardized column names and fixed data types.

3. **Feature Preprocessing**
   * **Categorical Encoding:** Applied One-Hot Encoding and Label Encoding.
   * **Feature Scaling:** Scaled numerical variables using `StandardScaler` / `MinMaxScaler`.
